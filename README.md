# EWU Lost & Found Portal - CSE302 Mini Project

**Description:**
A localized database web application built for East West University (EWU). It allows students to report lost items and enables campus security to manage and track found items. 

**Key Features:**
* **User Roles:** Student (can post and delete own reports) and Admin (can manage all reports and mark items as found).
* **CRUD Operations:** Supports Create (reporting items), Read (viewing dashboard), Update (marking as found), and Delete (removing resolved items).

**Setup Instructions:**
1. Start Apache and MySQL in XAMPP.
2. Create a database named `ewu_lost_found` in phpMyAdmin and run the `database.sql` code.
3. Place project files in the `htdocs` folder.
4. Run `http://localhost/ewu-project/` in your browser.
